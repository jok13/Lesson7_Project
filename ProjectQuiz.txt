What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.

A content delivery network is a system of servers that deliver web content to a user based on the location of the user, the origin of the webpage and the server. 
Some advantages are....
1. CDN files are hosted on different domains
2. CDN's offer higher avaliablity, lower network latency, lower packets.
3. It allows you to link specific CSS files and jQuery Librarys including the latest modified version.
